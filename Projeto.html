O que é HTML

Voltar ao inicio

HTML é a sigla de HyperText Markup Language, expressão inglesa que significa "Linguagem de Marcação de Hipertexto".

Consiste em uma linguagem de marcação utilizada para produção de páginas na web.

Permite a criação de documentos que podem ser lidos em praticamente qualquer tipo de computador e transmitidos pela internet.

Um documento em HTML é um texto simples, que pode ser editado no Bloco de Notas (Windows) ou Editor de Texto (Mac) e transformado em hipertexto.

A linguagem HTML foi criada por Tim Barners Lee na década de 1990. As especificações da linguagem são controladas pela W3C (World Wide Web Consortium).

Falando sobre Tags

Voltar ao inicio

As tags são formadas por uma estrutura própria, iniciam com o sinal “menor que”, em seguida vem o nome daquele elemento e por fim, o sinal “maior que”. Podem ser dispostas em tags que precisam de fechamento e tags que fecham sozinhas (self-closing).

O fechamento de uma tag será definido com uma barra (/), sendo que no caso das tags de auto fechamento, não há necessidade da presença desse caractere.

Atributos das Tags

Voltar ao inicio

Atributos HTML são palavras especiais usadas dentro da tag de abertura para controlar o comportamento do elemento.

Os atributos HTML são um modificador de um tipo de elemento HTML. Com eles podemos identificar melhor um elemento.

Informar qual arquivo aquela tag deve utilizar, indicar o tipo de um campo de texto, etc…

Há dois tipos de atributos no HTML, os globais que são aceitos por todas as tags, como por exemplo: class, id, lang, style etc...

E os específicos que somente algumas tags possuem, como src, disabled, href, label, etc…

Tags h

Tags h (h1, h2, h3, h4, h5, h6):

Definição e Uso:

As tags h1 a h6 são usadas para definir cabeçalhos HTML.

h1 define o cabeçalho mais importante. h6 define o título menos importante.

Nota: Use apenas um h1 por página - isso deve representar o cabeçalho/assunto principal de toda a página. Além disso, não pule os níveis de títulos - comece com h1, depois use h2, e assim por diante.

Tag p (parágrafo)

A tag 'p' define um parágrafo.

Os navegadores geralmente indentam a tag blockquote

Tag strong

Voltar ao inicio

A tag 'strong' é usada para definir o texto com forte importância. O conteúdo interno geralmente é exibido em negrito .

Exemplo:

Este texto é normal.

Este texto é importante!

Tag u

Voltar ao inicio

A tag 'u' representa algum texto não articulado e com estilo diferente do texto normal.

Marca palavras para destacar. O conteúdo interno geralmente é exibido com um sublinhado.

Exemplo:

Este é um texto que destaca esta palavra.

Tag i

Voltar ao inicio

A tag 'i' define uma parte do texto em uma voz ou humor alternativo. O conteúdo interno geralmente é exibido em itálico

A tag é frequentemente usada para indicar um termo técnico, uma frase de outro idioma, um pensamento, um nome de navio, etc.

Exemplo:

Lorem ipsum é o texto de preenchimento mais popular da história.

Tag mark

Voltar ao inicio

A tag 'mark' define o texto que deve ser marcado ou destacado.

Exemplo:

Todo mundo deveria aprender programação!!!

Listas

Voltar ao inicio

A tag 'ol' define uma lista ordenada. Uma lista ordenada pode ser numérica ou alfabética.

Exemplo:
  1. Carro
  2. Casa
  3. Moto
  4. Jetski

A tag 'ul' define uma lista não ordenada (marcada).

Use a tag 'ul' junto com a tag 'li' para criar listas não ordenadas..

Exemplo:
  • Arroz
  • Feijão
  • Macarrão
  • Carne

Links

Voltar ao inicio

A tag 'a' define um hiperlink, que é usado para vincular de uma página a outra.

O atributo mais importante da tag 'a' é o atributo 'href', que indica o destino do link.

Por padrão, os links aparecerão da seguinte maneira em todos os navegadores:

  • Um link não visitado é sublinhado e azul
  • link visitado é sublinhado e roxo
  • link ativo é sublinhado e vermelho

    Visit W3Schools.com!
